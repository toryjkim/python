# 데이터 타입 종류
- Boolean: True or False
- Numbers: 숫자
- String: 문자
- Bytes: 바이트 1010 2진수로 표기된 데이터
- Lists: [1, 2, 3]
- Tuples: (1, 2, 3) or 1, 2, 3 - 추가, 삭제, 수정 불가능한 list
- Sets: {1, 2, 3} - 중복 데이터 불가능
- Dictionaries: {'a':'A'} - key:value 형태의 자료구조

```
v_True = True
v_False = False
v_int = 1
v_float = 1.7
v_string = "Hello"
v_bytes = b"123"

v_list = [1, 2, 3]
v_tuple = 1, 2, 3 # (1, 2, 3)
v_typle = (1, 2, 3) # (1, 2, 3)
v_set = {1, 2, 2} # {1, 2}
v_dict = {
  'name': '123'
}
```
**note:** type 확인방법은 type(변수)
